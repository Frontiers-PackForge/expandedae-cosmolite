## Modos extras de bloqueio: Isso adiciona dois modos extras de bloqueio a todos os Fornecedores de Padrões
### Padrão: Este é o modo de bloqueio padrão do AE2, ele enviará o padrão se o armazenamento conectado não contiver nenhum dos insumos do padrão neste Fornecedor de Padrões. Se o armazenamento contiver qualquer coisa que não seja um insumo do padrão, o modo de bloqueio irá ignorá-la e ainda assim enviará o padrão.
    
![blocking_1.png](assets/blocking/blocking_1.png)

### Completo: Este modo de bloqueio não enviará o padrão se o armazenamento conectado contiver qualquer coisa.
    
![blocking_2.png](assets/blocking/blocking_2.png)

### Inteligente: Permite que o Fornecedor de Padrões envie o mesmo padrão se o armazenamento alvo contiver apenas os insumos desse padrão específico.

![blocking_3.png](assets/blocking/blocking_3.png)
